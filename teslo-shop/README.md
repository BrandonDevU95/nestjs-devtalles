## Description

**Teslo Shop** es una API RESTful de backend diseñada para una plataforma de comercio electrónico. Construida con **NestJS** y **PostgreSQL**, este proyecto sirve como el núcleo para gestionar productos, autenticación de usuarios y pedidos. Incluye una configuración robusta de base de datos contenerizada con **Docker** y administración visual mediante **pgAdmin**.

## Project setup

1.- Clonar el proyecto

```
git clone <repository-url>
```

2. - Clonar el archivo **.env.example** y renombrarlo a **.env**

3.- Configurar las variables de entorno en el archivo **.env**

4.- Ejecutar el siguiente comando para instalar las dependencias

```
pnpm install
```

5.- Levantar la base de datos de PostgreSQL y pgAdmin con docker-compose

```
docker-compose up -d
```

6.- Levantar la aplicación en modo desarrollo

```
pnpm run start:dev
```
